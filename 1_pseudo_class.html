CSS PSEUDO-CLASSES

the css pseduo-classes allow u to style the dynamic states of an elem such as hover,active and focus state,as well as elements that are existing in the document tree but cant be targeted via the use of other selectors without adding any ID's or classes to them,for eg., targeting the first or last child elements.

A pseudo-class starts with a colon (:).its synatx can be given with :

selector:pseudo-class{property:value;}
